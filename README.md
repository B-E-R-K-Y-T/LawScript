# Язык написания контрактов: LawScript!

Запуск:

```
# Windows
set PYTHONPATH=%CD%
py editor/app.py

# Linux/Mac
export PYTHONPATH=$(pwd)
python editor/app.py
```

Данный язык совмещает в себе две философии: Декларативную и Императивную
